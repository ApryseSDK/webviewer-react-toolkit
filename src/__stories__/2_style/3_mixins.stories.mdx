# Mixins

The WebViewer React Toolkit ships with some Sass mixins that wrap common
patterns. These can only be used in Sass files, but the code is provided below
so that you can implement them in CSS if you wish.

> For more on Sass mixins, check out the
> [Sass documentation](https://sass-lang.com/documentation/at-rules/mixin).

## Using

To use these in your own project, you can include the mixins in your code:

```scss
div {
  @include flex-center;
}
```

Or copy the code into your CSS files:

```css
div {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

If you are using Sass, make sure you import the Sass mixins file:

```scss
@import "~@pdftron/webviewer-react-toolkit/dist/scss/_mixins.scss";
```
