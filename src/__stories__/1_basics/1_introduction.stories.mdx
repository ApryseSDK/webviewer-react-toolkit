# WebViewer React Toolkit

The WebViewer React Toolkit is a React component library contains various
components that integrate with the
[PDFTron WebViewer API](https://www.pdftron.com/documentation/web/).

The goal of this toolkit is to create a set of highly customizable components
that take WebViewer documents (and other objects), and wrap them in specific
functionality.

## About the Documentation

This documentation is always kept up to date with the latest release. This may
be newer than the current stable release, as unstable releases are published
more frequently. The version of the toolkit that this documentation is for
is <Current />. This can be viewed at any time by hovering the PDFTron logo on
the top left.

When viewing <LinkTo title="Components/Button" story="Basic">Components</LinkTo>,
there will be a `Playground` tab on the top left. This provides a canvas where
you are able to interact with the component's props and view the generated code.
