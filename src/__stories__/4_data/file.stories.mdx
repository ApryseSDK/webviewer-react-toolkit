# File

The `File` class is a utility that encapsulates useful functionality from the
WebViewer API. It exposes various properties and methods for manipulating the
`File`.

```ts
const file = new File({ /* FileDetails */ });

await file.rotate();
setThumbnail(file.thumbnail.get());
```

## Providing a License Key

Since file uses [WebViewer APIs](https://www.pdftron.com/documentation/web/)
internally, you must provide a license key in order to remove watermarking and
enable the full functionality. This can be done by either setting a global
license, or individually including licenses for files during initialization:

```ts
// Globally set the license for all files (this is the best method)
File.setGlobalLicense('your-license-key');

// Individually set license for a file instance (this will take priority if set)
const file = new File({license: 'your-license-key'})
```
